# Particle System Renderer [![C99](https://img.shields.io/badge/C-00599C?logo=c&logoColor=white)](#) ![License](https://img.shields.io/github/license/vsaint1/c-particle-system.svg)

A minimal particle renderer with `CPU` and `GPU` support, designed for learning `graphics programming` and testing purposes.

## Features
- CPU particle simulation & rendering
- GPU particle simulation & rendering
- Written in `C99` and `OpenGL`
